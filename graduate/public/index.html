深度学习算法实验平台

一个帮你快速开始深度学习、监视和管理深度学习过程

方便使用

You can choose from a wide range of models, or create your own model based on basic operations.

It's extensible and customizable

Polyaxon supports by default a large number of highly modular built-in neural network layers, regularizers, optimizers, metrics... That being said it's easy to add your own operations.

It's open source

Polyaxon is an open source project, and it's based on Tensorflow and other open source projects. The application provides full Docker support.

How to start

Run the following: 
1 git clone https://github.com/polyaxon/polyaxon.git
2
3 cd polyaxon
4
5 docker/setup
6
